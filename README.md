# Jatan Rajbhar — Portfolio Website

A Facebook-style interactive portfolio website built with pure HTML, CSS, and JavaScript. Showcases professional experience, projects, skills, and achievements in a familiar social-media layout.

**Live:** [jatanrajbhar.vercel.app](https://jatanrajbhar.vercel.app/)

---

## Features

- **Facebook-style layout** — cover photo, profile card, tabbed navigation, and a live post feed
- **Post feed** — 9 chronological posts (Feb 2026 → Jun 2022) with color-coded type tags (New Role, Project, Education, Achievement)
- **Post interactions** — like, comment (inline composer), and share (copies URL to clipboard)
- **Post composer** — create new posts directly from the profile page
- **Global search** — filters posts in real time by keyword
- **Tabs** — Posts, About, Projects, Photos, each with dynamic fade-in animation
- **Open to Opportunities badge** — pulsing green indicator on the profile card
- **Downloadable resume** — links to `resume/jatan.pdf`
- **Responsive design** — works on desktop and mobile
